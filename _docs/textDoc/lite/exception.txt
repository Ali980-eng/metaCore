lite/exception.hpp
	A comprehensive exception class hierarchy for the lite library.
	Description
		The exception class hierarchy provides a structured and consistent way to handle errors in the lite library.
		The base class lite::exception serves as the foundation for all exceptions, while additional classes represent specific error conditions, such as internal library errors, outofrange access attempts, undefined behavior detection, wrong indexing operations, incorrect variable or type declarations, invalid object construction, and invalid input data.
