lite/micros.hpp
    Description
        This header file defines a set of macros and function prototypes for working with strings and lambda expressions in C++.
        It provides definitions for common string manipulations, lambda functions, and a way to print output.
    Macro Definitions
        nl: defined as \n, used for newline characters
        tab: defined as \t, used for tab characters
        object: defines the type of object being used (not explicitly shown in this code)
        Print(output): expands to std::cout << output; and prints the input string
        PrintLn(output): expands to std::cout << output << nl; and prints the input string followed by a newline character
        lambda(capture_clause, parameters, return_type, script): expands to a lambda function definition with capture clauses, parameters, return type, and script.
            The capture clauses are used to specify which variables in the function scope will be captured.
            The script is where the function body should go.
