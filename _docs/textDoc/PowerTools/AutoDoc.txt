AutoDoc.ps1
	A PowerShell script for generating documentation from source code.
	Description
		AutoDoc.ps1 is a script that uses Ollama to generate documentation for C/C++, Java, PowerShell, Python, C#, and JavaScript/TypeScript files.
		It supports multiple output formats including Doxygen, Markdown, and Plain Text.
	Functions
	Get-LanguageFromExtension
		Determines the programming language from a file extension.
		Parameters:
			$Extension (string)
		Returns:
			string (language code)
	Document-File
		Documents a single file by processing its content, language, and output format.
		Parameters:
			$FilePath (string)
			$OutputFormat (string)
			$OutputDir (string)
			$Force (bool)
		Returns: None
	Main Script
		Checks Ollama version, pulls a model, creates an output directory, and resolves the input path.
		It then documents files in the specified folder or recursively through subfolders.
	Parameters

		Parameter Description
		Path The file or folder path to document. Default: .\Main.cpp
		OutputFormat The desired output format (doxygen, markdown, plain text). Default: doxygen
		OutputDir The base directory for the output files. Default: Documentation
		Recurse Recursively process subfolders in the input path. Default: False
		Extensions An array of file extensions to consider when determining the language code. Default: (.cpp, .c, .h, .hpp, .java, .ps1, .psm1, .py, .cs, .js, .ts)
		Force Overwrite existing output files if they already exist. Default: False
		Model The name of the Ollama model to use. Default: llama3.2:1b
	Examples
		To document a single file, use:
		.\AutoDoc.ps1 -Path .\Main.cpp -OutputFormat doxygen
		To document files in all folders recursively through subfolders, use:
		.\AutoDoc.ps1 -Path .\src -OutputFormat All -Recurse -Extensions @('.cpp', '.h')
	Note: You need to install Ollama from its website: https://ollama.com
	Usage
		To run the script, navigate to the AutoDoc.ps1 file and use it as a script.
		You can specify the input path, output format, and other parameters using the parameters listed above.
		.\AutoDoc.ps1 -Path .\main.cpp -OutputFormat doxygen
	This will document the main.cpp file in the specified output directory with Doxygen format and language code for C++.
