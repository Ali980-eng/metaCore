Module PowerDoc.psm1
	This module generates documentation (TXT, MD, Doxygen, or All formats) for code files using Ollama.
	Functions
		Set-OllamaModel
			Set the Ollama model to use for documentation generation. This function changes the global model name used by Invoke-AutoDoc.
			Parameters:
				Name: The name of the Ollama model (e.g., "phi3:mini", "llama3.2:3b")
			Returns:
				None
		Invoke-AutoDoc
			Generate documentation for code files using Ollama.
			This function supports TXT, MD, Doxygen, or All formats and processes a single file or a folder.
			Parameters:
				Path: The path to a file or folder (optional)
				OutputFormat: The format of the output (e.g., 'txt', 'md', 'doxygen', or 'All' (generates all three))
				OutputDir: The directory where generated documentation will be saved
				Recurse: If Path is a folder, process all matching files recursively
				Extensions: File extensions to include (default includes common languages)
				Force: Overwrite existing output files
			Returns:
				None
		Document-File
			Document a single file using Ollama.
			This function generates Doxygen or Markdown output depending on the specified format.
			Parameters:
				FilePath: The path of the file to document
				OutputFormat: The format of the output (e.g., 'doxygen', 'md', or 'All' (generates all three))
				OutputDir: The directory where generated documentation will be saved
				Force: Overwrite existing output files
			Returns:
				None
		Document-File - Main Script Logic
			Check Ollama availability, check if the model exists and pull it if needed, create a base output directory, and resolve paths for single files or folders.
		Global Configuration
			Set the default Ollama model to "phi3:mini".
			This can be changed with Set-OllamaModel.
